**DOOCAL** is a high-performance, premium hyperlocal marketplace connecting customers with neighborhood vendors in real-time. Built with a "Mobile-First" philosophy, it combines the discovery power of **Google Maps** with the transactional ease of **Blinkit** and **Zomato**.

---

## üåü Key Pillars

### üõçÔ∏è For Customers: "Find it. Broadcast it. Get it."
*   **Hybrid Home Dashboard**: Experience a seamless transition between a curated list of top-rated local shops and an interactive real-time map.
*   **Contextual Catalog Requests**: Browse a vendor's digital menu/service list and send a request for a specific item in one tap.

## üöÄ Getting Started

### Prerequisites
*   Flutter SDK (3.x+)
*   Dart 3.x
*   A Firebase Project with Firestore and Auth enabled
*   Google Maps API Key

### Installation
1.  **Clone the Repo**
    ```bash
    git clone https://github.com/devang7072/DOOCAL--Hyperlocal-Product-Finder-.git
    ```
2.  **Install Dependencies**
    ```bash
    flutter pub get
    ```
3.  **Firebase Setup**
    *   Add your `google-services.json` (Android) and `GoogleService-Info.plist` (iOS).
    *   Enable Phone Authentication and Firestore.
4.  **Run the App**
    ```bash
    flutter run
    ```
